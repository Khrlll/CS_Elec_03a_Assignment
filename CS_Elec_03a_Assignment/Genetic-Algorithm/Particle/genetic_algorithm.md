Description: 
A Genetic Algorithm is an optimization technique inspired by natural evolution. It solves problems by creating a population of possible solutions and improving them over generations using:

*Selection (choose the best)
*Crossover (combine solutions)
*Mutation (random changes)

GA is useful when problems are too complex for exact solutions, especially when there are many possible combinations.

Examples:
1. Traveling Salesman Problem (TSP)
Goal: Find the shortest route visiting multiple cities.
GA tries different routes, keeps the shortest ones, and improves them over time.

2. Feature Selection in Machine Learning
Goal: Choose the best combination of features for a model.
GA evolves feature sets to improve accuracy.
